# FutureTech
FutureTech / JS + Vite + NPM + HTML + SCSS + BEM / Semantic and adaptive layout</br>
FutureTech / JS + Vite + NPM + HTML + SCSS + BEM / Семантический и адаптивный макет</br>

## JavaScript компоненты, реализованные в проекте:
1. Базовый абстрактный класс - основа для всех компонентов с автоматическим обновлением UI при изменении состояния.</br>
2. Утилита pxToRem - конвертация пикселей в REM-единицы для адаптивности.</br>
3. MatchMedia API - управление отзывчивым поведением на разных разрешениях экрана.</br>
4. Мобильное навигационное меню - выпадающее меню с бургер-кнопкой для мобильной версии.</br>
5. Табы - переключаемые вкладки с контентом.</br>
6. Кастомный селект - стилизованный выпадающий список.</br>
7. Кастомный видеоплеер - видеоплеер с собственным UI.</br>
8. Раскрывающийся контент - кнопка для полного раскрытия скрытого контента.</br>
9. Маска ввода телефона - форматирование номера телефона с помощью библиотеки inputMask.</br>

## Адаптивная вёрстка макета
Проект реализован с использованием HTML, препроцессора SASS (SCSS) и JavaScript с применением методологии БЭМ для организации классов и файловой структуры.</br>

Для управления зависимостями используется npm, а JavaScript-компоненты построены по модульной системе с использованием современных возможностей ES6+.</br>

Особенность проекта — полный отказ от пикселей в пользу относительных единиц измерения - rem.</br>
Для работы с размерами используются кастомные SASS-функции и миксины.</br>

Вёрстка полностью адаптивна и корректно отображается на различных устройствах и разрешениях экрана.</br>

## Установка
Чтобы запустить проект локально, выполните следующие шаги:

1. Клонируйте репозиторий:
  git clone https://github.com/Futtur1stt/FutureTech.git
2. Перейдите в директорию проекта:
  cd FutureTech
3. Установите зависимости:
  npm install
4. Запустите компиляцию SASS
  npm run dev
5. Откройте файл index.html в браузере

## Доступные команды

- `npm run dev` - Запуск development-сервера с горячей перезагрузкой
- `npm run build` - Сборка проекта для production в папку `dist`
- `npm run preview` - Предпросмотр собранного проекта
